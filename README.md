🚗 Car Finder India

A smart used car search tool built with Python and Streamlit.
Search through 13,000+ real Indian car listings based on your budget,
fuel type, transmission, and more.

## 🔗 Live App
[Click here to try it](https://carfinder-india.streamlit.app/)
## 🛠️ Built With
- Python
- Streamlit
- Pandas
- Real dataset from Kaggle (13,870 listings)

## ✨ Features
- Filter by budget, fuel type, transmission, owner type
- Search across 13,000+ real used car listings
- Multiple selection filters
- Clean, simple UI

## 📁 Project Structure
- `app.py` — Streamlit web interface
- `cars.py` — loads and cleans the dataset
- `filter.py` — filtering logic using OOP
- `display.py` — display formatting
- `main.py` — terminal version of the app
